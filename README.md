# Financial-Modelling-of-Coca-Cola-Consolidated-Inc.-COKE-
This project focuses on building a comprehensive financial model of The Coca-Cola Company using historical financial data. The objective was to analyze financial performance, forecast future growth, and estimate the company's intrinsic value using Discounted Cash Flow (DCF) valuation.
# Model Structure

- The Excel model includes:
Income Statement Forecast (5 Years)
Cash Flow Projections
Working Capital Assumptions
CapEx & Depreciation Estimation
Discount Rate (WACC) Calculation
DCF Valuation
Sensitivity Analysis

- Key Analysis Performed
Revenue growth trend analysis
Margin analysis (Gross, Operating, Net)
Free Cash Flow calculation
Intrinsic valuation using DCF
Sensitivity analysis on WACC and Growth Rate
